# Vedic-Ai-Guide-
This is my Vedic AI bot project! I built it using Google AI Studio, where I created prompts that explore insights from Vedic texts. While it’s not a traditional code-based bot, I’m showcasing the prompts and ideas here.
This Vedic AI bot is designed to help people understand complex shlokas, mantras, and deeper meanings in ancient scriptures. Many find these texts difficult to grasp, so I built this bot as a guide like a modern guru. It aims to simplify and reveal the deeper layers behind these ancient teachings, making them easier and more reliable to understand for everyone.
